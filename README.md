# react项目模板

### 目录说明

```
+ build                 // 编译后目录
+ lib                   // 第三方依赖
- app                  // 页面
    - component     // 页面级组件
    - index.html    // 测试页面
    - index.jsx     // 页面入口
    - index.scss    // 页面样式
.editorconfig
.eslintrc
.gitignore
package.json
README.md
webpack.config.js       // webpack配置
```

### 使用教程

```
tnpm run start
```
